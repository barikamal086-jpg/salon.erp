🚀 Guia de Implementação

Passo 1: Copiar CSS Variables

Adicione as CSS variables no início do seu index.html (dentro de <style> ou em um arquivo css separado).

Passo 2: Substituir Cores Hardcoded

No seu index.html, troque cores hardcoded por variáveis:

/* Antes */ background: #2563EB; /* Depois */ background: var(--primary);

Passo 3: Atualizar Componentes

Copie as classes CSS dos botões, cards, etc. do DESIGN_SYSTEM.html para seu index.html

Passo 4: Testes

✓ Abra este arquivo (DESIGN_SYSTEM.html) no navegador para ver todos os componentes
✓ Use como referência enquanto implementa no seu ERP
✓ Mantenha este arquivo atualizado quando fizer mudanças
